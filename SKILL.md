# IntercomX Agent Skill File

This application provides real-time crypto data using public APIs.

## Agent Capabilities

1. Fetch token price in USD
2. Fetch 24h price change
3. Fetch Ethereum mainnet gas fees
4. Fetch trending crypto tokens

## Available Functions

- getPrice()
- getGas()
- getTrending()

## Usage Context

Agents can call these functions to retrieve and display
live crypto market data for users.

This app does not require wallet connection.
